## Requirements:


- PHP ^8.1
- Node.js & npm
- Sqlite for local

## Project setup local

```shell
git clone git@github.com:AslanAV/php-project-57.git
cd php-project-57
make setup
```

## Project start local

```shell
make start
```

## Project setup and start docker

```shell
git clone git@github.com:AslanAV/php-project-57.git
cd php-project-57
make docker-setup-start-detached
```
